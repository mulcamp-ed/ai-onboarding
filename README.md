# 16기 AI 온보딩

SSAFY 16기 AI 온보딩 교육 자료 저장소입니다. Git · CLI · AI 도구 활용을 중심으로 Day01~Day05 일자별 교안, 가이드 문서, 실습 파일, 프로젝트 자료를 담고 있습니다.

각 일자 폴더는 대체로 다음 구조를 따릅니다.

- **강사용** — 강의용 교안 및 실습 코드
- **배포용** — 교육생에게 배포되는 가이드 문서 / 실습파일 / 프로젝트

---

      └─ 가이드문서/          # Render 활용 FastAPI 배포 가이드
