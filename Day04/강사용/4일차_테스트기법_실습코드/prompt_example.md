### 1. 바이브코딩으로 계산기 구현하기
```
<프롬프트>
HTML, JavaScript, CSS를 활용해서 두 개의 숫자를 입력받아 계산하는 심플한 계산기 페이지를 만들어줘.

<기능>
1. 첫 번째 숫자를 넣는 칸(input)과 두 번째 숫자를 넣는 칸(input), 그리고 [+] 버튼과 [-] 버튼을 만들어줘.
2. [+] 버튼을 누르면 두 숫자를 더해서 화면에 '더하기 결과는 OO입니다.'라고 출력해줘.
3. [-] 버튼을 누르면 첫 번째 숫자에서 두 번째 숫자를 빼서 화면에 '빼기 결과는 OO입니다.'라고 출력해줘.
```

### 2. 덧셈과 뺄셈 기능에 대한 단위 테스트 작성하기 
```
<프롬프트>
방금 추가한 기능(덧셈, 뺄셈)이 제대로 작동하는지 검증하는 유닛 테스트 코드를 작성해줘.

[조건]
- 외부 라이브러리(Jest 등)나 어려운 개발 개념은 절대 쓰지 말아줘.
- 기존에 만든 계산기 코드 밑에 테스트 결과를 볼 수 있도록, 하나의 파일에 이어서 작성해줘.
- 절대 기존에 만든 계산기 코드는 수정하지 마.
```

### 3. 새로운 기능을 추가하고, 기존 기능(덧셈, 뺄셈)이 잘 동작하는 지 확인하기
```
<프롬프트>
아래 코드에 새로운 기능을 추가해줘.

<기능>
기존 기능은 건들지 말고, 빼기 결과가 음수(< 0)가 되면 결과를 무조건 0으로 고쳐줘.
```

### 4. 일부로 기존 기능을 망가뜨려서 테스트해보기
```
<프롬프트>
테스트 코드는 수정하지 말고, 기존 더하기 기능을 실행하면 최종 결괏값에 +1을 한 값을 출력하도록 코드를 수정해줘.
```
